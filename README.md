# 🚀 AWS Exam Prep Hub

A modern, interactive web app to prepare for AWS certifications with **MCQ-based practice, instant feedback, and smart revision tools**.

🌐 **Live at:**
https://pratik-p-crest.github.io/AWS-Exam-Prep/

---

## ✨ Features

### 🧠 Quiz Mode

* Randomized MCQ questions
* Topic-based filtering
* After Quiz answer feedback
* Progress tracking (correct / wrong / skipped)

### 📖 Revision Mode

* Review all questions anytime
* Filter:

  * ✅ Answered
  * ❌ Wrong
  * 🆕 New
* Search questions instantly
* Track learning progress

### 📊 Performance Insights

* Score calculation
* Accuracy tracking
* Topic-wise understanding

## 🚀 How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/pratik-p-crest/AWS-Exam-Prep.git
   ```

2. Open folder:

   ```bash
   cd AWS-Exam-Prep
   ```

3. Run:

   * Simply open `index.html` in your browser
